# 퀴즈 시스템 공공데이터 연동 기획

> v2.0 — 2026.04.07 업데이트 (1단계 완료 반영)
> 공공 API 데이터 수집 → AI 가공 → 퀴즈 대량 생성.

---

## 배경

- 초기 퀴즈 30개 수동 삽입 → 하루 10개 배정이면 3일에 고갈
- 목표: 공공데이터 기반 대량 확보 (1단계) + 뉴스 기반 무한 생성 (2단계)
- 컨셉: **"데이터는 공공 API, 포장은 AI"** → 할루시네이션 리스크 제로
- 정답/팩트는 공공데이터 원본, AI는 오답 보기 생성 + 설명 다듬기만 담당

## 1단계: 공공 API 데이터 수집 — ✅ 완료

### 소스 A. 재경부 시사경제퀴즈 — ✅ 157개 삽입 (2026.04.07)

| 항목 | 내용 |
|---|---|
| 형태 | CSV 파일 (EUC-KR 인코딩, 콤마 구분) |
| URL | https://www.data.go.kr/data/15131287/fileData.do |
| 원본 | 200문항 (OX + 객관식 혼합) |
| 처리 결과 | 157개 삽입, 43개 비금융 문제 필터링 |
| 스크립트 | `scripts/process-moef-csv.ts` |
| 처리 내용 | CSV 파싱 → Claude API로 톤 조절 (반말/친근체) + 비금융 필터링 → DB 삽입 |

### 소스 B. 한국은행 ECOS 경제용어 — ✅ 306개 삽입 (2026.04.07)

| 항목 | 내용 |
|---|---|
| 엔드포인트 | `https://ecos.bok.or.kr/api/StatisticWord/{key}/json/kr/{start}/{end}/{word}` |
| 인증 | ECOS API 키 (`ECOS_API_KEY`) |
| 수집 방식 | 한글 음절 + 경제 키워드 + 영문자별 검색으로 전체 수집 |
| 수집 결과 | 306개 용어 (중복 제거 후) |
| 스크립트 | `scripts/fetch-ecos-words.ts` → `scripts/generate-quizzes.ts` |
| 처리 내용 | 용어+정의 수집 → Claude API로 객관식 퀴즈 변환 (정답 원본, 오답 AI 생성) → DB 삽입 |

## 2단계: 뉴스 기반 AI 퀴즈 자동 생성 — 🔜 미진행

### 목표

- 매일 아침 크론잡으로 경제 뉴스 수집 → AI 퀴즈 10개 자동 생성
- 퀴즈 고갈 완전 해소

### 네이버 뉴스 검색 API

| 항목 | 내용 |
|---|---|
| 인증 | 네이버 개발자센터 앱 등록 (Client ID/Secret) |
| 호출제한 | 25,000회/일 |
| 방식 | "금융", "경제", "투자" 키워드로 뉴스 수집 |

### 자동 생성 플로우

```
크론잡 (매일 아침 07:00):
  1. 네이버 뉴스 API → 오늘의 경제 뉴스 수집
  2. 뉴스 본문에서 핵심 키워드/팩트 추출
  3. Claude API로 퀴즈 10개 생성:
     - 정답: 뉴스 속 팩트 (확정)
     - 오답 보기: AI가 그럴듯한 오답 3개 생성
     - 설명: 뉴스 원문 기반 요약
  4. quizzes 테이블에 insert
  5. source = "오늘의 경제뉴스" 태깅
```

## 퀴즈 배정 로직

### 현재 방식 (유지)

- 랜덤 셔플 + exclude 방식
- 오답노트 복습 30% + 새 퀴즈 70%
- `quiz.service.ts`의 `getTodayQuizzes()` 참고

### 고갈 체크

- 순번(seq) 기반이 아닌 **유저별 풀은 퀴즈 수 / 전체 퀴즈 수**로 체크
- 뉴스 크론잡이 매일 10개씩 추가하므로 실질적 고갈 불가

## 스크립트 목록

| 스크립트 | 설명 | npm 명령어 |
|---|---|---|
| `scripts/process-moef-csv.ts` | 재경부 CSV → 톤 조절 → JSON | `npm run quiz:process-moef` |
| `scripts/fetch-ecos-words.ts` | ECOS 경제용어 수집 → JSON | `npm run quiz:fetch-ecos` |
| `scripts/fetch-seibro-terms.ts` | 예탁결제원 금융용어 수집 → JSON | `npm run quiz:fetch-seibro` |
| `scripts/fetch-moef-quiz.ts` | 재경부 API 수집 (CSV 대신 사용 시) | `npm run quiz:fetch-moef` |
| `scripts/generate-quizzes.ts` | 수집 용어 → AI 퀴즈 변환 → JSON | `npm run quiz:generate` |
| `scripts/bulk-insert-quizzes.ts` | JSON → DB 벌크 삽입 | `npm run quiz:insert` |

### 실행 방법

```bash
# 재경부 CSV 처리 (톤 조절 + DB 삽입은 Supabase CLI)
npm run quiz:process-moef
supabase db query --linked -f data/moef-insert.sql

# ECOS 경제용어 (수집 → AI 변환 → DB 삽입)
npm run quiz:fetch-ecos
npm run quiz:generate
supabase db query --linked -f data/ecos-insert.sql

# 예탁결제원 (DATA_GO_KR_API_KEY 필요)
npm run quiz:fetch-seibro
npm run quiz:generate
supabase db query --linked -f data/seibro-insert.sql
```
