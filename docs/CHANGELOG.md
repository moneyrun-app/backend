## [v2.0] 2026-04-02 — MVP 대폭 축소, 3페이지 집중

### 변경 문서 전체

| 문서 | 변경 요약 |
|---|---|
| `01_프로덕트_기획서` | 5탭 → 3페이지. 커뮤니티·자산트래킹·시뮬레이션 제거. 페이스메이커+마이북 집중 |
| `02_API_통합_명세서` | 23개 엔드포인트 확정. 커뮤니티/시뮬레이션 API 제거 |
| `03_전체_플로우_정의서` | 유저 여정·일일 사이클·마이북 흐름 재작성 |
| `07_기술결정사항` | MVP 축소, 온보딩 간소화, 등급 기준 변경 등 6건 기록 |

### 주요 변경 내역

- 온보딩: 7문항 → 4스텝 (나이, 실수령액, 좋은소비, 고정소비)
- 등급 판정: 연금 시뮬레이션 → 저축 비율 기반 (RED/YELLOW/GREEN)
- 페이스메이커: 하루 1회 AI 생성 + DB 캐싱
- 잉여자금: 역산 방식 (실수령액 - 좋은소비 - 고정소비)
- 마이북: 상세 리포트(월1회 자동), 주간 리포트(유저 입력), 학습 콘텐츠, 스크랩
