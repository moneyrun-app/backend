## 2. sections 구조

`sections`는 9개 객체 배열. 각 객체의 `section` 키로 분기해서 렌더링.

```
sections[0].section === "A"  →  재무 건강 진단
sections[1].section === "B"  →  돈의 흐름
sections[2].section === "C"  →  통합 시뮬레이션
sections[3].section === "D"  →  한국의 현실
sections[4].section === "E"  →  등급별 로드맵
sections[5].section === "F"  →  잉여자금 늘리기
sections[6].section === "G"  →  금융 교육
sections[7].section === "H"  →  12개월 캘린더
sections[8].section === "I"  →  금융 용어 사전 (선물)
```

`step` 값으로 페이지/탭 그룹핑:

```
step 1 = "나의 '돈 관리' 성적표"  →  Section A, B
step 2 = "이대로 가면 만나게 될 나"  →  Section C
step 3 = "머니런과 달려갈 새로운 미래"  →  Section D, E, F, G, H
step 0 = "선물"  →  Section I
```

## 3. 섹션별 렌더링 가이드

### Section A: 재무 건강 진단

```json
{
  "section": "A",
  "step": 1,
  "title": "재무 건강 진단",
  "totalScore": 67,
  "maxScore": 100,
  "grade": "RED",
  "scores": [
    { "axis": "지출 비율", "score": 5, "max": 20 },
    { "axis": "잉여자금 비율", "score": 11, "max": 20 },
    { "axis": "고정비 유연성", "score": 16, "max": 20 },
    { "axis": "변동비 여유도", "score": 15, "max": 20 },
    { "axis": "투자 가용 기간", "score": 20, "max": 20 }
  ],
  "peerComparison": {
    "ageGroup": "30대",
    "expenseRatio": { "user": 83, "peer": 60 },
    "surplusRatio": { "user": 17, "peer": 22 },
    "variableRatio": { "user": 33, "peer": 28 }
  },
  "chart": {
    "type": "radar",
    "labels": ["지출 비율", "잉여자금", "고정비 유연성", "변동비 여유도", "투자 기간"],
    "data": [5, 11, 16, 15, 20]
  },
  "ai_narrative": "시은아, 너의 재무 건강 점수는 100점 만점에 67점이야..."
}
```

**렌더링:**
- 상단: 큰 원형 점수 (67/100) + 등급 뱃지 (RED)
- 중앙: **레이더 차트** (5축, chart.labels + chart.data)
- 각 축별 점수 바 (score/max)
- 또래 비교 카드 (peerComparison → "30대 또래 평균" vs "나")
- 하단: ai_narrative 텍스트

**추천 차트 라이브러리:** recharts `<RadarChart>` 또는 chart.js radar

## 4. 공통 사항

### 금액 포맷

```
원 → 만/억 단위로 변환
500000 → "50만원"
35000000 → "3,500만원"
239283000 → "2.4억원"
```

### 등급 색상

```
RED    → #EF4444 (빨강)
YELLOW → #FBBF24 (노랑)
GREEN  → #34D399 (초록)
```

### 차트 라이브러리 추천

- recharts (React)
- chart.js (범용)
- 타임라인/게이지는 커스텀 SVG 또는 CSS

### 페이지 구성 추천

```
[헤더] 시뮬레이터 분석 리포트 | 등급 뱃지 | 날짜

[탭 또는 스크롤]
  Step 1: 나의 성적표 → Section A, B
  Step 2: 미래 시뮬레이션 → Section C
  Step 3: 새로운 미래 → Section D, E, F, G, H
  선물 → Section I

[하단] disclaimer 면책 문구
```

### ai_narrative 렌더링

- 일반 텍스트 (마크다운 아님)
- 줄바꿈은 `\n`으로 구분될 수 있음
- 폰트 크기 본문보다 약간 크게, 읽기 편한 줄간격
- 배경색 살짝 다르게 (말풍선 느낌)
