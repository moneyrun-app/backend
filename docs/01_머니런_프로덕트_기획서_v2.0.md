## 2. 서비스 비전 & 핵심 메시지

> **"부자 되세요"가 아니라 "거지 되지 마세요"**

복잡한 투자·시뮬레이션 대신, 매일 AI가 한마디씩 잔소리해주는 구조. 유저가 매일 앱을 열면 "오늘 너 얼마 쓸 수 있어", "이번 주 이거 좀 많이 썼다?", "이 글 읽어봐" 같은 메시지를 받는다.

핵심은 **습관**이다. 한 번 진단하고 끝나는 게 아니라, 매일 돌아와서 페이스메이커 메시지를 확인하는 루틴을 만드는 것.

## 3. 타겟 유저

20대 중후반 ~ 30대 초반 사회초년생. 돈을 벌기 시작했지만 어떻게 관리해야 하는지 모르는 사람. "투자 앱은 무섭고, 가계부는 귀찮은" 사람.

| 소득 구간 분류 | 커뮤니티와 함께 추후 |

## 5. 로그인 정책 & 온보딩

### 로그인

카카오 소셜 로그인 단일. 비로그인 상태는 없음 — 앱 진입 시 바로 로그인.

### 온보딩 (로그인 직후, 최초 1회)

간단한 정보 입력 → 잉여자금(하루 사용 가능 금액) 즉시 계산.

**입력 항목:**

| 순서 | 항목 | 입력 방식 | 필드명 |
|---|---|---|---|
| 1 | 현재 나이 | 숫자 입력 (만 나이) | `age` |
| 2 | 월 실수령액 | 숫자 입력 (원 단위) | `monthlyIncome` |
| 3 | 좋은 소비 | 항목별 금액 입력 | `goodSpendings[]` |
| 4 | 고정 소비 | 항목별 금액 입력 | `fixedExpenses` |

**좋은 소비 항목:**

| 코드 | 라벨 | 설명 |
|---|---|---|
| `savings` | 적금 | 은행 적금 |
| `investment` | 투자 | 주식·ETF 등 |
| `pension_savings` | 연금저축 | 연금저축펀드 등 |
| `irp` | IRP | 개인형퇴직연금 |
| `insurance` | 보험 | 보험료 |

**고정 소비 항목:**

| 코드 | 라벨 | 설명 |
|---|---|---|
| `rent` | 월세 | 0원 가능 (부모님 집 등) |
| `utilities` | 관리비+공과금 | |
| `phone` | 통신비 | |

**온보딩 결과:**

```
잉여자금 = 월 실수령액 - 좋은 소비 합계 - 고정 소비 합계
하루 사용 가능 = 잉여자금 / 30

→ "너 하루에 OO만 원 쓸 수 있어"
```

온보딩 완료 즉시 → 페이스메이커(홈)로 이동.

## 6. 전체 서비스 구조

### 3페이지 구조

```
[카카오 로그인]
  → [온보딩] (최초 1회)
  → [페이스메이커(홈)] ← 메인 화면, 매일 돌아오는 곳

하단 3탭:
  페이스메이커  |  마이북  |  마이
```

| 탭 | 라우트 | 핵심 역할 |
|---|---|---|
| **페이스메이커** | `/home` | AI 일일 메시지 + 하루 사용 가능 금액 + 오늘의 한마디 |
| **마이북** | `/book` | 상세 리포트 / 주간 리포트 / 스크랩 / 학습 콘텐츠 (카테고리별, 날짜별) |
| **마이** | `/my` | 내 정보 수정 + 계정 설정 |

## 7. 잉여자금 자동 계산 시스템

### 핵심 공식

```
좋은 소비 합계 = Σ(goodSpendings[].amount)
고정 소비 합계 = rent + utilities + phone

잉여자금(월) = monthlyIncome - 좋은 소비 합계 - 고정 소비 합계
잉여자금(주) = 잉여자금(월) / 4.3
잉여자금(일) = 잉여자금(월) / 30
```

### 재계산 트리거

마이페이지에서 어떤 항목이든 수정하면 → 즉시 전체 재계산.

| 수정 항목 | 재계산되는 값 |
|---|---|
| 월 실수령액 | 잉여자금 전체, 일/주/월 사용 가능 금액 |
| 좋은 소비 | 잉여자금 전체, 일/주/월 사용 가능 금액 |
| 고정 소비 | 잉여자금 전체, 일/주/월 사용 가능 금액 |
| 나이 | 페이스메이커 참조 데이터 (연금 관련 메시지) |

## 8. 페이스메이커 (홈)

### 정의

AI가 먼저 말을 거는 구조. 유저가 물어봐야 답하는 챗봇이 아니라, **매일 1번 메시지를 생성**해서 보여준다. 앱을 열면 바로 보이는 메인 화면.

### 표시 정보

1. **오늘의 AI 메시지** (핵심, 화면 상단 고정)
2. 하루 사용 가능 금액 ("오늘 OO원 쓸 수 있어")
3. 신호등 등급 뱃지

### AI 메시지 생성 — 참조 데이터

| 참조 데이터 | 출처 | 예시 |
|---|---|---|
| 상세 리포트 | 마이북 > 상세 리포트 | "지난주 식비가 평균보다 많았네" |
| 주간 리포트 | 마이북 > 주간 리포트 | "이번 주 잘 버텼다" |
| 유저 스크랩 | 마이북 > 스크랩 | "너 비상금 글 스크랩했잖아, 실천하고 있어?" |
| 운영 상수 데이터 | system_config | "서울 평균 월세가 73만 원인데 너는..." |
| 유저 재무 프로필 | 온보딩 데이터 | 나이, 실수령액, 좋은 소비, 잉여자금 |
| 오늘 날짜/요일 | 시스템 | "월요일이니까 이번 주 예산 다시 세워볼까?" |

### 메시지 톤 가이드

- **찐친이 잔소리** — 걱정하면서도 직접적으로 말하는 톤
- 모든 메시지 끝은 **행동 유도** ("이거 해봐", "이거 읽어봐", "이거 생각해봐")
- 등급별 톤 차이:

| 등급 | 톤 | 예시 |
|---|---|---|
| RED | 직접적 + 유머 | "야 하루에 OO원밖에 못 쓰는데 배달 또 시켜? ㅋㅋ" |
| YELLOW | 은근히 찌르기 | "저축 하고 있긴 한데... 이 속도면 좀 빡빡하다?" |
| GREEN | 격려 + 다음 단계 | "오 꽤 잘하고 있네. 이거 한번 읽어봐 →" |

### 메시지 갱신 주기

- **하루 1번** — 매일 자정(KST) 기준으로 새 메시지 생성
- 같은 날 다시 열어도 같은 메시지
- 백엔드에서 AI(Claude API)로 생성 → DB 저장 → 프론트에서 조회

### 메시지 하단 — 추천 행동

AI 메시지 아래에 1~2개 추천 행동 표시. 클릭 시 마이북 해당 콘텐츠로 이동.

## 9. 마이북

### 정의

내 금융 관련 콘텐츠가 모이는 공간. 카테고리별로 정리되고, 날짜별로 쌓인다.

### 카테고리 구조

| 카테고리 | 코드 | 설명 |
|---|---|---|
| **상세 리포트** | `detailed_report` | AI가 생성한 내 재무 분석 리포트 (주기적 자동 생성) |
| **주간 리포트** | `weekly_report` | 매주 자동 생성되는 한 주 요약 |
| **스크랩** | `scrap` | 유저가 직접 스크랩한 콘텐츠 모음 |
| **금융 학습** | `learn` | 운영팀이 등록한 등급별 금융 교육 콘텐츠 |

### 상세 리포트

- AI(Claude)가 유저 데이터를 분석해서 작성하는 맞춤형 리포트
- **생성 트리거:** 재무 프로필 변경 시
- **제한:** 유저당 **월 1회만** 생성 가능 (마지막 생성일 기준 30일 경과 필요)
- 페이스메이커가 리포트 내용을 참조하여 일일 메시지에 반영

### 주간 리포트

- **유저 입력 기반** — 유저가 "이번 주 상태"를 간단히 입력
- AI가 한 쪽짜리 가이드 페이지 생성

### 스크랩

- 금융 학습 콘텐츠 중 유저가 스크랩한 것만 모아서 표시
- 페이스메이커가 참조

### 금융 학습

- 운영팀이 등록하는 등급별 맞춤 콘텐츠
- 1분 안에 읽을 수 있는 분량
- 읽음 처리 / 스크랩 가능

**MVP 학습 콘텐츠 (초안):**

| 등급 | 제목 |
|---|---|
| RED | "비상금 없으면 진짜 거지 됩니다" |
| RED | "배달비 월 30만 원 = 1년 360만 원" |
| RED | "복리가 뭔지 모르면 평생 월급쟁이" |
| YELLOW | "청년도약계좌 안 하면 정부가 주는 돈 버리는 거" |
| YELLOW | "연말정산 환급 적은 이유 99% 이거" |
| GREEN | "연금저축 지금 안 하면 65세에 후회" |
| GREEN | "ETF 모르면 적금만 하다 인플레이션에 짐" |

## 11. 신호등 등급 시스템

### 등급 판정 기준

```
좋은 소비 비율 = 좋은 소비 합계 / 월 실수령액 * 100

RED    : 좋은 소비 = 0원 OR 비율 < 10%
YELLOW : 10% <= 비율 < 20%
GREEN  : 비율 >= 20%
```

### 등급 역할

등급은 페이스메이커 톤과 마이북 학습 콘텐츠 추천에 사용된다.
